14 // There is one copy of each of the Redistributor registers per PE.
15 // An operating system executing at Non-secure EL1 uses either the GICC_* or the GICV_* registers to control interrupts, and is unaware of the difference.

17 //The CPU interface handles physical interrupts at all implemented Exception levels:
18 //• Interrupts that are translated into LPIs are optionally routed via the ITS to the Redistributor and the CPU
19 //interface.
20 //• PPIs are routed directly from the source to the local Redistributor.
21 //• SPIs are routed from the source through the Distributor to the target Redistributor and the associated CPU
22 //interface.
23 //• SGIs are generated by software through the CPU interface and Redistributor. They are then routed through
24 //the Distributor to one or more target Redistributors and the associated CPU interfaces
